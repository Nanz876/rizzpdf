Common questions

Do my files get uploaded anywhere?
Never. Every tool runs entirely in your browser using JavaScript. Your files don't touch our servers.
What's the difference between Day Pass and Pro?
Day Pass is a $1 one-time unlock that lasts 24 hours — great if you have a batch to process today. Pro is $5/month and adds 200MB file support, file history, and priority support with no time limit.
Can I cancel Pro any time?
Yes — cancel any time from your dashboard. No questions asked, no cancellation fee.
What counts as a "file" on the free tier?
3 files per tool per browser session. So you can merge 3 PDFs, compress 3 PDFs, and convert 3 PDFs — all free in the same session.
