Three approaches — pick the scope that feels right

A

Full Replica — Homepage + Navbar + Tool Pages + Footer

Everything gets the iLovePDF treatment. New tool-grid homepage, ALL CAPS nav, iLovePDF-style tool pages (centered title + big upload button), dark multi-column footer. Most complete, most work.

B

Homepage + Navbar Only — Tool pages stay as-is

Replace the homepage with the tool grid and update the navbar. Individual tool pages keep their current design for now. Fast, high-impact, minimal risk to working tools.

app/page.tsx Navbar.tsx
C

Full Replica — Recommended ⭐

Same as A but done in two phases. Phase 1: homepage + navbar (ships fast, looks great immediately). Phase 2: tool pages + footer. Each phase is a clean deploy.

Phase 1 → ships immediately Phase 2 → follow-up
